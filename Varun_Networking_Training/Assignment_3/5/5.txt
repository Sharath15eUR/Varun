First step is to identify the given network and subnet mask

In this question, the given Network Address is 192.168.1.0/24 and we need 4 subnets
Original subnet mask = /24 (255.255.255.0)

Number of bits to be borrowed for subnetting : 2 (since 2^2 = 4 subnets)

Therefore, new Subnet mask = /26 ==> 255.255.255.192

To find each subnet block size : Block size = 256 - 192 = 64 
i.e, Each subnet increments by 64
